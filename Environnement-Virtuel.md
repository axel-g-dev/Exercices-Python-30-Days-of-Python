### ✅ **Créer un environnement virtuel**

```bash
python3 -m venv env
```

### ✅ **Activer l’environnement**

```bash
source env/bin/activate
```

### ✅ **Installer des packages (si besoin)**

```bash
pip install nom_du_package
```

### ✅ **Lancer le programme**

```bash
python3 mon_programme.py
```

### ✅ **Désactiver l’environnement**

```bash
deactivate
```
